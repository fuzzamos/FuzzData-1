### Fuzzing workflows a fuzz job from start to finish


[文章出处](https://foxglovesecurity.com/2016/03/15/fuzzing-workflows-a-fuzz-job-from-start-to-finish/)

1. 是否存在样例代码
2. 是否能够自己编译
3. 是否存在可用的测试样例
